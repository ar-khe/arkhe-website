✨ Phonotactics

This is very much a work in progress, phonotactics will be defined at the same time as morphology and script, in order to make them as compatible as possible. But for the moment;

Syllable Structure:

(C1(C2/S))V(V2)(C3)

  • C1: Any consonant
  • C2: If C1 is a non-glottal plosive, then C2 can be /s/,/l/,/ʂ/
  • S: A semivowel, either /j/ or /w/
  • V: Vowel of the syllable, can be long or short. If there is an S, V can’t be long
  • V2: If V is short and there isnt an S, V2 can be any other short vowel
  • C3: Can be /n/, /m/, /s/, /ɸ/, /ʂ/, /l/
Moraic Structure:

Idk if I’ll leave this in, but for the moment:

  • (C1(C2/S))V is one mora
  • (V2) and (C3) are one mora each
